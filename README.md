# Email_Client_API

Um die Semesterarbeit in Programmieren zu bearbeiten habe ich mir als Trigger die Kursschwankung von Bitcoin rausgesucht.

Der Triger soll ausgelösst werden wenn der BTC Preis innerhalb von um:
1 h = - 1% oder + 1%
24h = -3% oder +3%
Oder ob der BTC über oer unter eine festgelegte Grenze bewegt.
(Es soll halbstündlich geprüft werden)

Auf den Trigger hin soll eine Idividuelle Email an mich gesendet werden.
Mit den Nachrichten.
Das es Bewegung im Markt gab und Daten angehängt.

Als extra:

#Verschlüsselung
Die Passwörter sollen in einer geschützen Datei liegen.

#Daily Kypro & Stock Update
Das Programm soll mich jeden morgen über meine Stocks & BTC in einer Mailinformieren.
